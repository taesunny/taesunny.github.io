# Solution

``` java
class Solution {    
    public long solution(int w, int h) {
        int gcd = getGcd(w, h);
        
        return (long)w*h - ((w/gcd)+(h/gcd)-1)*(long)gcd;
    }
    
    public int getGcd(int w, int h) {
        if(h == 0) {
            return w;
        }
        
        return getGcd(h, w%h);
    }
}
```
