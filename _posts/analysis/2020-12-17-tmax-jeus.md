## JEUS
- 국내 TMAX Soft사의 WAS(Web Application Server)
- Java EE 7 표준을 준수
- Java Enterprise-User Solution의 약자

## JEUS8 주요 표준 지원 사항
- Enterprise Application
    - EJB
    - JMS
    - CDI
    - Bean Validation
    - Managed Beans
    - DependencyInjection
- Web App
    - Servlet
    - JSP
- Resource
    - JDBC
